Build a polished, playable 3D Gradius game built with Three.js.

Requirements:

- Pure client-side. The finished product must be a static site: an index.html at the repo root (or in dist/ if you use a build step) that works with no server-side code.
- Your own original visual style — do not copy Gradius exactly.
- Polish counts: game feel, difficulty curve, and juice. Sound is optional but welcome.

You have one shot: when you stop, whatever is in the repo is the submission.
