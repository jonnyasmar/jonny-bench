Build a playable polyphonic synthesizer as a static web app.

Requirements:
- Pure client-side using the Web Audio API — no audio sample files. The finished product must be a static site: an index.html at the repo root (or in dist/ if you use a build step) that works with no server-side code.
- On-screen piano keyboard, at least 2 octaves, playable by mouse/touch AND the computer keyboard.
- At least 2 oscillator waveforms, ADSR envelope controls, a filter with cutoff + resonance, and a delay or reverb effect.
- Make it look and feel like an instrument you'd actually want to play.

You have one shot: when you stop, whatever is in the repo is the submission.
